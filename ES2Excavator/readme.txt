Earthsiege 2 Vol file unpacker.
3/25/2024

Requirements:
	+ You will need to have the Java runtime installed on the target machine.
	+ Extract the jar and bat / sh file to the dir with your VOL files.
	+ run .bat or .sh file.
		+ enter name of vol file with Extension.
		+ Any path should be relative to where the JAR is.

Outcome:
	+ console prompt will show output.
	+ Vol file is unpacked to a directory relative to where JAR is.
		+ example: SIMVOL0.VOL -> simvol/simvol0/
